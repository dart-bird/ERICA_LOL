# Install JAVA

1. `sudo apt-get update`
2. `sudo apt-get install default-jdk`

# HOW TO

## ★ required java version 11.0.5

1. `git clone https://github.com/krumaska/ERICA_LOL`  
2. `cd ERICA_LOL/leagueofjava`  
3. `javac Main.java`
4. `java Main`
5. `Enjoy Playing`
